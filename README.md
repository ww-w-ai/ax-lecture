# ax-lecture

**AI 시대의 바이브코딩과 에이전트 활용법** — 누구나 보고 쓰는 강연·학습 자료 공개 저장소.

> AI 시대, 혼자서 팀처럼: 만들고(bkit) · 자동화하고(에이전트 수지) · 자율주행시키기(Sprint).
> LLM이 어떻게 답을 만드는지(추론 원리)부터 실전 도구 활용까지.

이 자료는 **KAIST에서 1인 창업가를 위해 진행된 강연 내용을 정리해 공개**한 것입니다. 주제 자체는 청중을 가리지 않으므로 누구나 보고 활용할 수 있도록 일반화했습니다.

## 누구를 위한 자료인가
- **이 저장소**: AI 코딩·에이전트를 더 잘 다루고 싶은 **누구나** (개발자·기획자·창업가·학생·교육자).
- **원 강연**: 한 회차는 1인 창업가를 대상으로 진행됐으나, 내용 자체는 청중을 가리지 않는다.
- 권장 배경: Claude Code 사용 경험이 있으면 더 깊이 와닿음 (필수는 아님).

## 자료 구성 철학
- **슬라이드 = 요약·개념** (발표용), **핸드아웃 = 상세 레퍼런스** (배포·자습용).
- 슬라이드 홀짝 포맷: 발표 시 홀수(사진/도식)만 투사해 청중이 의미를 추측 → 강사가 구두 설명 / 배포 시 홀수+짝수로 자습.
